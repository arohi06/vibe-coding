# 🧠 Vibe Coding – Brain Dump To‑Do List

A productivity app designed for **daily brain dumps**, **priority setting**, and **calendar visualization**.  
Instead of juggling sticky notes or scattered thoughts, this project helps you capture everything in one place and see it clearly on a calendar.

---

## ✨ Features
- **Brain Dump Mode** → Quickly jot down raw thoughts, tasks, or ideas without worrying about structure.
- **Daily Priority Setting** → Mark tasks as 🔥 High, ⚡ Medium, or 🌱 Low priority.
- **Calendar View** → See all tasks mapped to dates for a clear timeline.
- **Task Components** → Modular UI for adding, editing, and tracking tasks.
- **Reflection Section** → Space to review progress and note personal insights.
- **Weekly Overview** → Visualize tasks across the week for better planning.

---

## 🚀 Tech Stack
- **React + TypeScript** → Core framework for building UI.
- **TailwindCSS** → Fast, responsive styling.
- **Vite** → Lightning‑fast bundler for development.
- **Custom Hooks** → For task tracking and toast notifications.
- **Calendar & UI Components** → Built with reusable TypeScript components.
